# Email Sequences by Tier

## tier_1 — 3-step sequence

Send on: day 0 / day 3 / day 7

Step 0 (day 0): Warm intro — specific pain point, one question.
Step 1 (day 3): Short follow-up — reference the first email, one direct question.
Step 2 (day 7): Final message — acknowledge no reply, leave door open, no pressure.

Rules:
- Subject lines: lowercase, under 8 words, no punctuation at end
- Body: plain text, 3 sentences max before CTA
- Never repeat the same angle twice in the sequence
- Step 2 must close the loop ("dernier message de ma part")

---

## tier_2 — 2-step sequence

Send on: day 0 / day 5

Step 0 (day 0): Direct, brief intro — one sentence on the value, one question.
Step 1 (day 5): Single follow-up — softer tone, open the door if timing was wrong.

Rules:
- Even shorter than tier_1 (max 2 sentences before CTA)
- No third email — let it breathe

---

## nurture — 1 email only

Send on: day 0

One email only. No follow-up.
Save the sequence for when they respond or re-engage.

Use angle: mid_team by default (see SIGNALS.md for alternatives).

---

## General sequence rules

- Never send more than one email per day to the same contact.
- If a contact replies at any point: pause the sequence immediately (manual review).
- Subject lines for follow-ups: "suite — [company]" or "relance — [company]"
- Never use "Je me permets de vous contacter" as an opener.
- Sign-off: first name only. No title, no phone number, no logo.
